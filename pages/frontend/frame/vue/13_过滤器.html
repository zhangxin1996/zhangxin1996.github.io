# 定义过滤器

Vue允许你自定义过滤器，可被用于一些常见的文本格式化。过滤器可以用在两个地方：双大括号插值和v-bind表达式（后者从2.1.0+开始支持）。过滤器应该添加在JavaScript表达式的尾部，由”管道“符号指示：

<!-- 在双大括号中 -->
<h2>{{message | capitalize}}</h2>

<!-- 在v-bind中 -->
<h2 v-bind:id="rawId | formatId"></h2>

# 全局过滤器和局部过滤器

# 全局过滤器

Vue.filter("capitalize", function(value, data1, data2) {
  // 返回处理后的值
})
1
2
3

在创建 Vue 实例之前全局定义过滤器，filter方法有两个参数，参数一是“capitalize”是这个过滤器的名字；参数二是个匿名函数，匿名函数中参数一是过滤器管道符前面传过来的数据，参数二（data1）和参数三（data2）是过滤器名字后传递的参数。

也可以使用ES6的箭头函数：

Vue.filter("capitalize", (value, data1, data2) => {
  // 返回处理后的值
})
1
2
3

了解了如何定义全局过滤器，以下进行具体操作：

<body>
  <div id="app">
    <h2>{{message | capitalize}}</h2>
  </div>

  <script src="../js/vue.js"></script>
  <script>
    Vue.filter("capitalize", value => {
      if(!value) return "";
      let val = value.toString();
      return val.slice(0, 1).toUpperCase() + val.slice(1);
    })

    const app = new Vue({
      el: "#app",
      data: {
        message: "hello",
      }
    });
  </script>
</body>

# 私有过滤器

私有过滤器也叫局部过滤器，实现是在Vue实例对象中，以下是具体操作：

<body>
  <div id="app">
    <h2>{{message | capitalize}}</h2>
  </div>

  <script src="../js/vue.js"></script>
  <script>
    const app = new Vue({
      el: "#app",
      data: {
        message: "hello",
      },
      filters: {
        capitalize(value) {
          if(!value) return "";
          let val = value.toString();
          return val.slice(0, 1).toUpperCase() + val.slice(1);
        }
      }
    });
  </script>
</body>

# 过滤器的串联使用

{{message | filterA | filterB}}
1

在这个例子中，filterA被定义为接收单个参数的过滤器函数，表达式message的值将作为参数传递到函数中。然后继续调用同样接收单个参数的过滤器函数filterB，将filterA的结果传递到filterB中作为参数。

{{message | filterA("arg1", arg2)}}
1

filterA是接受三个参数的过滤器函数，其中message的值作为第一个参数，普通的字符串"arg1"作为第二个参数，表示式arg2的值作为第三个参数。

下面实现一个过滤器串联的案例：

<body>
  <div id="app">
    {{ message | msgFormat("疯狂+1") | test("，呦呦呦！") }}
  </div>

  <script src="../js/vue.js"></script>
  <script>
    const app = new Vue({
      el: "#app",
      data: {
        message: "曾经，我也是一个单纯的少年，单纯的我，傻傻的问，谁是这世界上最单纯的男人"
      },
      filters: {
        msgFormat(value, data1) {
          return value.replace(/单纯/g, data1);
        },
        test(value, data1) {
          return value + data1;
        }
      }
    })
  </script>
</body>
