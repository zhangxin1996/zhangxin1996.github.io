# const的使用

# const的作用

const用于声明常量，何为常量即为值不可修改的量。

# const声明格式

const PI = 3.14;
1

# const声明常量的特性

  1. 一定要赋初始值

    const A;
    1

  2. 一般常量使用大写（潜规则的）

    const NAME = "kebe";
    1

  3. 常量一旦被声明，无法修改

    const NAME = "kobe";
NAME = "james";
    1
    2

  4. 具有块级作用域

    {
  const NAME = "kobe";
}
console.log(NAME);
    1
    2
    3
    4

  5. 对于数组和对象中的元素进行修改，不算对常量进行修改，不会报错

    原因是：常量所指向的引用类型的地址并没有发生变化；

    const NAMES = ["kebe", "james", "lilei"];
NAMES.push("coderz");
console.log(NAMES);

const STUDENTS = {
  name: "coderz",
  age: 18,
  height: 1.88
}
STUDENTS.name = "lucy";
console.log(STUDENTS);
